ㅤMODELO TCP/IP

O modelo TCP/IP (Transmission Control Protocol/Internet Protocol) é um modelo mais simples e prático para comunicação de rede do que o modelo OSI.

  • Ele consiste em 4 camadas: Acesso à Rede, Internet, Transporte e Aplicação.

  • O modelo TCP/IP é usado para comunicação na internet e em outras redes baseadas em IP.

    ㅤMODELO OSI

    O modelo OSI (Open Systems Interconnection) é um framework conceitual que descreve como os dados são transmitidos e recebidos em uma rede.

  • Ele consiste em 7 camadas, cada uma com seus próprios conjuntos de protocolos e funções.

  • As camadas são: Física, Enlace de Dados, Rede, Transporte, Sessão, Apresentação e Aplicação.

    ㅤSEGURANÇA E ACESSO

    DNS (Domain Name System) é um protocolo usado para traduzir nomes de domínio (como google.com) em endereços IP (como 74.125.224.72).

    Os servidores DNS mantêm um banco de dados de nomes de domínio e seus endereços IP correspondentes.

    ㅤSERVIDORES E A COMUNICAÇÃO

    Um servidor web é um programa que serve páginas da web para clientes sob demanda.

  • Os servidores web usam HTTP ou HTTPS para transmitir páginas da web aos clientes.
  • O HTTP é usado para transmitir páginas da web, enquanto o HTTPS é usado para transmitir informações sensíveis, como credenciais de login e números de cartão de crédito.
  • O UDP é usado para aplicativos que requerem comunicação rápida e de baixa latência, como jogos online, streaming de vídeo e VoIP.
